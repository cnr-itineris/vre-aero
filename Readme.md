# VRE-AERO

This repository contains a collection of routines developed for VRE-AERO powered by D4science available [here](https://itineris.d4science.org).

The guide which explain each code and its feature is available under VRE-AERO workspace as pdf file (corresponding to final Deliverable 8.15).

Note: several routines make use of external API services which can raise exceptions or other duty interruptions.
